# Teoria do Caos: Atraidor de Lorenz

Este projeto apresenta uma simulação matemática do Atraidor de Lorenz desenvolvida em GNU Octave. O objetivo é visualizar o comportamento caótico de sistemas dinâmicos, popularmente conhecido como "Efeito Borboleta".

## O que é o Atraidor de Lorenz?
É um conjunto de soluções caóticas do sistema de equações de Lorenz. Ele demonstra como condições iniciais quase idênticas podem levar a resultados completamente diferentes, criando uma estrutura visual complexa em 3D que nunca se repete.
